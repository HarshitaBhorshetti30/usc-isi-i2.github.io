ABOUT

In today’s big data era, there are tremendous opportunities for exploiting available information to solve many kinds of problems by building empirical, data-driven models of complex phenomena. Yet, building such models is generally an elaborate process, often requiring a data scientist trained in the requisite computer science technologies. Subject matter experts (SMEs) have the domain knowledge required to define useful empirical models, but often lack the computer science expertise to construct them from scratch. Today, creating empirical models requires a SME/data-scientist team, making production and refinement of models expensive and time consuming.

“Data Scientist in a Box” (DSBox) technologies captures the expertise of a data scientist in a computer system. DSBox innovations address the key bottlenecks in the creation of empirical models. Data preparation is known to take 60% to 80% of the effort to build models: DSBox proposes technology that uses machine learning to profile a dataset and to automatically detect and correct all the common errors that data scientists fix today using a variety of tools and scripts. Creation of appropriate features is known to be the key to create accurate empirical models: DSBox proposes a mathematically general approach for constructing useful and informative features in a domain-agnostic way. Deep learning has shown large performance improvements in some application domains: DSBox proposes new algorithms to make deep learning more broadly applicable. Composition of machine learning pipelines is difficult because there are many alternative ways of featurizing data and many alternative machine learning algorithms with varying performance characteristics: DSBox proposes technology that learns from prior expert-grade solutions, encoding them in a knowledge base, and using them to automatically create working models with minimal effort and to refine them to optimize their performance.
